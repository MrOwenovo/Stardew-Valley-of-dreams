【MOD內容簡介】﻿
Q：這個MOD是做什麼的？
這是個以NPC生日為主題的對話擴充包！
對話的觸發條件非常簡單：在NPC家人朋友生日的前一天跟NPC對話（有例外，詳見註1），以及在生日當天跟壽星本人對話。

（註1：海莉除外：生日前一天的春13由於是節日，相關的親友台詞都會提前到春12日。跟艾米莉結婚的話依舊是春13觸發對海莉的特殊台詞。）


Q：就這樣？
是的！不過所有對話都設置了隨機內容或者問答選項，所以每年對話都會有不同的內容（應該）。
部分隨著好感度而態度大幅變化的角色（點名謝恩、萊納斯和喬治爺爺），我也設置了不同的對話差分。
而且所有可婚NPC的台詞都會隨著交往進度而有所變化。婚後的生日對話，伴侶和他們的家人或許會把一些心底話告訴你哦。
另外，幾組異性曖昧NPC之間也互有對話，但是默認是關閉的。在選項裡可以設置打開，詳見後文的選項說明。

Q：還有其他功能嗎？
為了讓生日更有沉浸感一些，我為生日時的一些活動也增加了對話。
【行程對話】部分NPC生日前後的行程有些小對話，不過屬於彩蛋級別，錯過了也沒有什麼損失。遊戲裡如果找到的話就當是小小的驚喜吧！
【電影院對話】生日當天邀請NPC會有特殊台詞！可婚的NPC會有【朋友/交往/婚後】三個版本（部分角色有訂婚版本，但也是彩蛋級別，不需要特別去刷），而可婚NPC的家人會有【婚前/婚後】兩個版本。大部分角色都有兩種以上的台詞，所以盡情帶你喜歡的NPC去看生日電影吧！他們不會當覆讀機的（應該）。
另外對於電影的喜好，在生日當天會提升1級（不喜歡->喜歡；喜歡->最愛）。會有此等級提升是因為我覺得就算你不喜歡這部電影，生日時有人請你看電影應該還是挺開心的吧XD
【慶祝配偶生日的對話】你（終於）可以對配偶說生日快樂啦！包括12名可婚NPC和克羅布斯，一共有3種選項：【普通祝賀/祝賀時順便說些情話/表示你根本忘了對方的生日】，全部都不影響好感度，請隨意選擇。

小小的擴展閱讀：
【送禮對話】生日禮物的特殊台詞不包含在此MOD裡，但可以考慮下載我的另一個MOD【更獨特的收禮反應】，裡面包括了NPC對所有最愛的生日禮物的特殊台詞。


Q：文本量是多少？
為34位村民都增加了全套的對話，我自己的統計大概有8萬字的文本量。應該夠玩好一陣子的。
另外由於模組性質，一些獨行獨往又無家室的NPC台詞會很少。主要文本集中在有朋友和有家庭的角色上。

Q：有離婚/劈腿/家暴相關的劇情嗎？
呃，沒有，因為我不玩這種玩法（笑哭）。對話的前提都是不劈腿不外遇不家暴不離婚的，所以單配偶且與配偶關係良好的話會有最佳的遊玩體驗。
我有想過做家暴的差分，最後還是下不了手。這是個溫馨向的模組。
但多配偶／劈腿的玩法下仍然可以正常使用，儘管劇情上或許會有些矛盾。
只是部分涉及兩個可婚NPC的劇情我設置了要求朋友狀態才能觸發。就是例如你婚了艾米莉，又跟海莉外遇的話，海莉對艾米莉的生日特殊台詞就會消失了。





【安裝說明/兼容性/選項說明】﻿

Q：前置MOD都有哪些？
【必須前置MOD】
SMAPI
Content Patcher
Event Repeater
（主要影響部分選項問答在明年會否重覆發生。不安裝的話部分問答在一個存檔裡只會發生一次。）

【建議】
Generic Mod Config Menu
（用來在遊戲裡調整MOD選項。手動調整Config.json也能有同樣效果）


Q：跟其他MOD兼容性如何？
此MOD標定了NPC在具體日子的對話（例：海莉覆寫了春14生日當天，及春26艾米莉生日前夕），如果其他MOD覆寫了同一天的話，Content Patcher會優先顯示名字順序靠前的MOD。但不會造成遊戲崩潰。
我想會覆寫同樣位置的台詞應該也是生日相關的台詞，所以請自行修改文件的檔案名來決定要顯示哪個MOD。（不過我好像還沒看到會標定生日前夕的對話MOD，所以應該不會衝突太多）

另外行程相關的對話都是基於原版的NPC行程來製作的，如果有安裝修改NPC行程的MOD，或許會出現對話跟實際行程不符的劇情bug。如果你有安裝修改NPC行程的MOD，可以在選項裡選擇關掉【Schedule Edit】來關掉此MOD的行程修改。


Q：現有存檔也能用嗎？
可以的！不過可婚角色的生日台詞很多都是三個階段互相關聯的，所以最佳體驗應該是從朋友階段開始玩。
但用在現有存檔也沒有任何問題，還沒有到非要開新檔去體驗的地步。

本MOD的可調節選項如下：

默認全關閉。
啟用後會有關於對方生日的台詞。上述五對都是獨立的開關。
基本上除了海莉和亞歷克斯明確過是互相有好感外，其他都是普通的朋友/鄰居/同事關係。但海莉和亞歷克斯也不是在交往的狀態……我自己是感覺他們只是互有戀愛方面的好感，啥都沒做，所以也往這方向寫了。
跟任何一方結婚後都會有對話變化，沒結婚的那方都會友好祝福玩家的婚姻。
這些對話的觸發條件是【跟兩方的NPC都是朋友關係】，或者跟【任意一方是婚姻關係】。跟玩家交往後，NPC們都會自動跟他們的曖昧對象斷絕來往，因為我喜歡有節操的星露谷 :D

我認為這部分的內容應該不會讓玩家覺得NPC之間在發展戀愛關係，但以防萬一仍然將這部分的內容設置成全！關！閉！遊玩前請按自己的喜好設置打開。
其實我沒有準備太多的內容，所以打不打開都不影響遊玩體驗。我只是因為覺得哈維和莉婭在鎮上基本沒有朋友，滿可憐的，才做了這個部分。
請注意，由於遊戲機制關係，打開後需要到遊戲時間的第二天才會發生台詞。請在生日前先想好要不要打開。


2. ［DialogueDisable］（停用某些對話）
默認打開，我也推薦打開。
主要停用了台詞發生日的雨天對話，以及格斯的酒館對話。
【雨天對話】如果不停用雨天對話，下雨天時村民有50%的概率使用原版的雨天對話，而不是顯示此MOD追加的生日對話。因此我停用了雨天對話（只有當天會有特殊台詞的村民受影響，其他村民會照常說出雨天對話）。
【格斯的酒館對話】格斯在原遊戲裡預設了位於酒館時只會反覆說同一句台詞，而他除了周一周二都不會離開酒館，所以如果不停用，格斯不會說任何生日相關的台詞。
同樣由於遊戲機制關係，打開後需要到遊戲時間的第二天才會生效。


3. ［Schedule Edit］（行程修改）
默認打開，如果不是出現MOD衝突，我也不建議關閉。
為了增加行程相關的台詞，我重寫了部分NPC的行程代碼（但沒有進行實質改動；只是加入了文本）。
因此如果有安裝更改NPC行程的MOD，在生日當天NPC或許會受此MOD影響而使用原版的行程。
我不知道具體哪些MOD對哪些行程作出了改動，但如果遇到了bug或者行程矛盾，可以選擇關閉此功能。（關閉此選項會導致此MOD增加的部分行程台詞不顯示，但這部分不是此MOD的主要內容，所以衝突了就關掉吧）
